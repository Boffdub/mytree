# MyTree

A climate trivia quiz app built with React Native and Expo. Answer questions about climate topics to grow your virtual tree!

## Description

MyTree is an educational mobile app that helps users learn about climate change through interactive quizzes. Users can:
- Answer questions across multiple categories (Energy, Transportation, Food & Agriculture, Carbon Removal)
- Grow a virtual tree as they progress
- View their tree and track their learning journey

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Expo CLI (installed globally or via npx)
- Expo Go app on your mobile device (for testing on physical devices)

## Getting Started
