# MyTree

A climate trivia quiz app built with React Native and Expo. Answer questions about climate topics to grow your virtual tree!

## What is MyTree?

MyTree helps users learn about climate change through interactive quizzes. As you answer questions correctly, you grow a virtual tree representing your learning journey.

**Categories:** Energy · Transportation · Food & Agriculture · Carbon Removal

## Technology Stack

- **React Native** + **Expo** (SDK 54) — mobile app framework
- **Supabase** — auth (magic link + Google OAuth), Postgres database, Edge Functions
- **AsyncStorage** — local guest mode persistence
- **React Navigation** — screen routing
- **Expo Linear Gradient** — UI components
